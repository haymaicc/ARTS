# 什么是ARTS？
ARTS是由[左耳朵耗子](http://weibo.com/haoel?s=6cm7D0)在其专栏[左耳听风](https://time.geekbang.org/column/48)发起的计划

每周通过这四个指标来训练自己：
* Algorithm：解决一道算法
* Review：点评一篇英文文章
* Tips：学习一项技术技巧
* Share：分享一个观点或思考

# 针对自己的ARTS
* Algorithm：[leetcode](https://github.com/azl397985856/leetcode)
* Review：[weekly](https://github.com/ruanyf/weekly)

* Share：最近阅读内容
